Polecenia NPM

  • npm init - inicjuje npm w projekcie i tworzy plik package.json
  • npm install - instaluje wszystkie zależności wymienione w package.json
  • npm list --depth=0 - wyświetla w terminalu listę lokalnie zainstalowanych pakietów wraz z numerami ich wersji, bez zależności
  • npm install [package-name] - instaluje pakiet lokalnie w folderze node_modules
  • npm uninstall [package-name] - odinstaluje pakiet zainstalowany lokalnie i zaktualizuje package.json
  • npm start i npm test - uruchomi skrypt start lub test, znajdujący się w package.json
  • npm run [custom-script] - uruchomi niestandardowy skrypt znajdujący się w package.json
  • npm outdated - używany do wyszukiwania aktualizacji, programowego wykrywania kompatybilnych wersji i wyświetlania listy dostępnych aktualizacji
  • npm update - zaktualizuje wszystkie pakiety do maksymalnej dozwolonej wersji
  • npm cache clean - czyszczenie pamieci podrecznej
