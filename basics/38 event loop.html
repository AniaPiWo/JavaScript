EVENT LOOP JS jest jezykiem jednowatkowym czyli NIE URUCHOMI jednoczesnie dwoch kawalkow kodu (dlatego nieskonczona petla while(true){}) zawiesi przegladarke) Zdarzenia (np klikniecie) nie beda przetworzone dopoki petla nie zostanie przerwana. EngineJS czeka na zakonczenie petli (co nigdy nie nastapi) i nie zajmie sie innymi zadaniami z kolejki zadan. JS jest synchroniczny, wykonuje kod linijka po linijce, przez co jego wydajnosc jest ograniczona i nie nadaje sie do pracochlonnych zadan (np przetwarzanie obrazow). Mozna sprawic, aby kod przetworzyl sie asynchronicznie dzieki funkcji setTimeout() (funkcja wywola sie po okreslonym czasie)
